Mycroft, a Clojure inspector

Key elements include:

  • Browse Namespaces
  • Browse Vars in a Namespace
  • Browse a Var that is a Fn
  • Browse some Vars that are data

Everything is made of data and functions:

  • Browse JMX Beans
  • Browse recent history
  • Browse Interactive REPL result
  • Browse Java things

Everything is thread safe.
